Projects

Number plate recognition system

This Java machine learning project leverages OpenCV to develop a car number plate recognition system. The system takes an image of a car as input and processes it through multiple stages. First, Otsu's thresholding method is applied to binarize the image, enhancing the contrast between the number plate and the background. Next, active contouring is used to accurately segment the number plate from the image. Finally, Tesseract OCR is employed to extract and predict the alphanumeric characters from the segmented number plate. This approach offers a robust solution for automated vehicle identification through number plate recognition.
